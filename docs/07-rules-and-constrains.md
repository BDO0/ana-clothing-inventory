# System Rules

## Rule 1
Stock is NEVER stored.

## Rule 2
Only events are stored for inventory.

## Rule 3
No deletion of events allowed.

## Rule 4
All events must have UUID + timestamp.

## Rule 5
Offline-first is mandatory.

## Rule 6
App must work fully without internet.

## Rule 7
Sync is background-only and optional.

## Rule 8
Cloud is backup only, not source of truth.
